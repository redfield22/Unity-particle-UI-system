# Unity Particle UI System

A Unity project created to learn and experiment with Unity's UI system and Particle System.

## Features

- UI Button Selection
- Multiple Particle Effects
- Simple UI Navigation
- Particle Simulation

## Technologies

- Unity
- C#
- Git
- GitHub

## Project Structure

```
Assets/
Packages/
ProjectSettings/
```

## Getting Started

1. Clone the repository

```bash
git clone <repository-url>
```

2. Open Unity Hub

3. Add/Open the cloned project

4. Open the main scene

5. Press Play

## Learning Goals

This project is focused on learning:

- Unity UI
- Unity Particle System
- Scene Management
- Git & GitHub workflow
